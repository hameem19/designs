 Builder Design Pattern is a Creational Design Pattern. Builder pattern builds a complex object using simple objects and using a step by step approach
  
1). java.lang.StringBuilder#append() (unsynchronized)
2). java.lang.StringBuffer#append() (synchronized)
3). java.nio.ByteBuffer#put() (also on CharBuffer, ShortBuffer, IntBuffer, LongBuffer, FloatBuffer and DoubleBuffer)
4). javax.swing.GroupLayout.Group#addComponent()
5). All implementations of java.lang.Appendable
